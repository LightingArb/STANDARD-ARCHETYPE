# STARDARD-ARCHETYPE

> AI 專案作業系統。一個 repo 管全部。

## 結構

```
├── 專案管理/                ← SRI 全部在這裡
│   ├── 指令.md              ← 第二層：人+AI 協作入口
│   ├── 大絕招.md            ← 第三層：Harness 自主推進入口
│   ├── 專案管理使用手冊.md   ← 第一層：無 AI 也能跑的結構規範
│   ├── 人格映射表.md        ← 支援層：SRI ↔ Furnace 對接
│   ├── 99_基地/
│   │   └── sa/              ← 6 份 sri-*.md
│   └── XX_專案名稱/         ← 專案（新增後出現）
├── 熔爐百相/                ← Furnace：AI 人格鍛造工坊
├── README.md
└── .gitignore
```

## 怎麼開始

1. Clone 這個 repo
2. 對 AI 說：讀 `專案管理/指令.md`，新增專案 [名稱]
3. 開始工作

## 三層架構

| 層級 | 入口 | 用途 |
|------|------|------|
| 第一層 | 專案管理/專案管理使用手冊.md | 結構規範（人類也能跑） |
| 第二層 | 專案管理/指令.md | 人+AI 協作流程 |
| 第三層 | 專案管理/大絕招.md | AI 自主推進（Harness） |
