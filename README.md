## 三態

| 態 | 入口 | 用途 |
|----|------|------|
| 純人工 | `專案管理/_系統/基地/專案管理使用手冊.md` | 手動照結構運作 |
| 指令驅動半自動 | `專案管理/指令.md` | 人類下指令，AI 執行 |
| 閘門式自主推進（Harness） | `專案管理/大絕招.md` | AI 自主推進，人類守硬閘門 |

## 怎麼開始

1. 先看 `專案管理/指令.md`
2. 建立第一個專案：`開一般專案 <名稱>` 或 `開程式專案 <名稱>`
3. 日常用 `討論專案`，需要自治推進時用 `啟動 Harness`

## 核心分工

- SRI：結構、規則、流程、閘門
- Furnace：人格卡與角色卡供應層
- MCP：外部工具層，設定在 `專案管理/_系統/基地/mcp/`
