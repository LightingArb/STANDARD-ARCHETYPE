# 04_主專案程式

## 這是什麼
這裡放可執行、可交付、可部署的執行單元，供對沖交易系統主線施工。

## 模組清單

- `001_前端頁面__探索`：快速驗證儀表板資訊架構與操作流程。
- `002_API服務__子專案`：負責行情、下單、帳戶與策略 API。
- `003_完整系統__整合`：負責把前端、API、策略與風控串成單一啟動流程。
- `A_壓力測試`：驗證高頻行情與告警情境下的系統穩定度。

## 整合計畫

- 第一步：先定義 `002_API服務__子專案` 的 REST 與事件契約。
- 第二步：`001_前端頁面__探索` 依契約做假資料介面。
- 第三步：`003_完整系統__整合` 負責串 Docker Compose、觀測與風控開關。
