```
┌─────────────────────────────────────────────┐
│            Layer 1：資料抓取層                │
│                                             │
│  觀測（3 個）           預報（19 個模型）      │

│  ├── WU (爬蟲)              Historical       │

                   ▼

│            Layer 2：偏差分析層                │

                   ▼
┌─────────────────────────────────────────────┐
│            Layer 3：校正與合併層              │

                   ▼

│            Layer 4：機率轉換層                │

                   ▼
```

## 核心原則

1. **方法論是結論，數字不是** — 所有具體參數由數據決定
2. **不做主觀預判** — 先全部抓再用數據篩選
3. **偏差穩定性 > 準確性** — 穩定的偏差可校正，不穩定的丟掉
4. **期望值驅動一切** — EV > 0 才進場
