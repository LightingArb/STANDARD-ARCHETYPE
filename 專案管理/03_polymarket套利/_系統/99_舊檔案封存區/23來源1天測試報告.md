**解析方法：**
1. Playwright 載入頁面，等 Angular 渲染（wait_for_selector + sleep 5s）

3. Table 0 第 2 行 = High Temp（結算值）
4. Table 1 欄位 index 1 = Temperature，用 regex 取數字
5. 溫度單位是 °F，需轉 °C
